# Privacy Policy

`development-plugin-for-avalonia` is a documentation and workflow plugin for Codex.

## Data Collection

This repository and plugin do not include a hosted backend, telemetry service, or account system operated by the plugin author.

The plugin package itself does not intentionally collect, store, or transmit personal data beyond whatever Codex, GitHub, or other tools you choose to use while working with it already process under their own policies.

## Third-Party Services

When you use Codex, GitHub, or any external services referenced by this repository, those services process data according to their own terms and privacy policies.

## Local Files

The plugin is designed to read and help edit files in the workspace where you use Codex. Any local file access is initiated by your Codex session and subject to your Codex configuration and approvals.
